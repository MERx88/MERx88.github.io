## 2024_11_27

### 솔리디티 라이브러리

**특징**

- 변수 선언 불가능
- 이더 송수신 불가능

**왜 씀?**

말그대로 라이브러리 이다. 작성된 코드를 도서관 책꺼내듯이 사용하는 라이브러리 조금 더 특징적인걸 설명하자면

`잘짜여진 코드 사용가능하다`
가스비가 컨트랙트에서 중요한데 내가 고민할필요없이 효율적인 코드를 사용할수있다 그외에 당연한거지만 언급하면 오류 발생률이 낮아지고 안정성, 보안성 시간 절약등이 있다.
