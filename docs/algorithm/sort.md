배열을 처음부터 돌면서 둘씩 비교하며 큰녀석을 계속 오른쪽으로 보내는 정렬

## 👆 Selection Sort 선택 정렬🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴

오름차순 정렬 기준일 때 배열을 돌면서 가장 작은 녀석을 찾고 이 녀석을 가장 작은 인덱스에 꽂아 넣으면서 하나씩 인덱스를 높여가며 계속 그다음 작은걸 꽂아넣는 방식

당연히 O(n^2) 이 된다.
